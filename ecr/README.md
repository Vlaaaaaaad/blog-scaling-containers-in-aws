# ECR

Files to create an ECR in `us-east-1` and `eu-west-1`.

## Providers

| Name   | Version       |
|--------|---------------|
| aws.eu | ~> 2.0 ~> 2.0 |
| aws.us | ~> 2.0 ~> 2.0 |

## Inputs

No input.

## Outputs

| Name           | Description                          |
|----------------|--------------------------------------|
| repo\_name\_eu | Repository URL for the eu-west-1 ECR |
| repo\_name\_us | Repository URL for the us-east-1 ECR |
